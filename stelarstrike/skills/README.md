# StelarStrike Skills

Skills are knowledge bases used by agents to understand and perform security testing techniques.
Each skill directory contains methodology documentation, payload lists, and example files.

## Available Skills

| Skill | Description |
|-------|-------------|
| SQL Injection | Union-based, blind, error-based, time-based SQLi across MySQL/PostgreSQL/MSSQL/SQLite/Oracle |
| XSS Injection | Reflected, stored, DOM XSS — filter bypass, polyglots, WAF bypass, CSP bypass |
| File Inclusion | LFI/RFI, wrappers, LFI-to-RCE techniques |
| Cross-Site Request Forgery | CSRF bypass techniques and cheat sheets |
| Insecure Direct Object References | IDOR enumeration and exploitation |
| Web Cache Deception | Cache poisoning and deception techniques |
